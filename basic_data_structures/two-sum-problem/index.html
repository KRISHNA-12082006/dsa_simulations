Two Sum – Visual Algorithm Simulator

Question — Samasya Kya Hai?

Humare paas ek array of integers hai aur ek target integer hai. Humein array mein se do numbers dhundhne hain jinki sum exactly target ke barabar ho.

Problem Statement:

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

Constraints:

  • Each input has exactly one solution
  • Same element cannot be used twice
  • Answer can be returned in any order

Example 1:

Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9

Example 2:

Input: nums = [3, 2, 4], target = 6
Output: [1, 2]
Explanation: nums[1] + nums[2] = 2 + 4 = 6

Approach — Hash Map (Optimal Solution)

Brute Force (Naive Approach)

Sabse pehla tarika jo dimag mein aata hai wo hai nested loops use karna:

  • Har element ke liye, uske baad wale saare elements check karo
  • Agar koi pair mil jaye jiska sum = target, to return karo
  • Time Complexity: O(n²) — bahut slow for large arrays
  • Space Complexity: O(1)

Optimal Approach: Hash Map

Hum ek hash map (dictionary) use karenge jo already dekhe hue numbers ko store karega:

Algorithm:

  1. Ek empty hash map banao
  2. Array ko traverse karo (har element ke liye):
    • Calculate complement = target - current_number
    • Check karo: kya ye complement pehle se hash map mein hai?
      • Agar hai: Match mil gaya! Return karo [complement_index, current_index]
      • Agar nahi: Current number ko hash map mein store karo

Why Hash Map?

  • Hash map mein lookup operation O(1) time mein hoti hai (average case)
  • Hum ek hi pass mein solution dhundh sakte hain
  • Previously seen elements ko efficiently track kar sakte hain

Complexity Analysis:

  • Time Complexity: O(n) — sirf ek baar array traverse karna hai
  • Space Complexity: O(n) — worst case mein saare elements hash map mein store ho sakte hain

Key Insight: Jab hum current element dekh rahe hain, hum ye nahi soch rahe ki "iske saath kaun add hoga", balki ye soch rahe hain ki "kya wo number jo iske saath add hokar target banayega, pehle aa chuka hai?"
