Problem Statement

Given two strings s and t of the same length, and a set of character transformations with associated costs, find the minimum total cost to make both strings identical. If impossible, return -1.

Your Input:
s = "abcc"
t = "bccc"
transformations = [['a','b'], ['b','c'], ['c','a']]
costs = [2, 1, 4]

Strategy

For each position i, we can transform both s[i] and t[i] to any common target character. The key is finding the target that minimizes: cost(s[i]→target) + cost(t[i]→target)

Transformation Graph

Each character is a node, and each transformation is a directed edge with a cost.

Shortest Path Distance Matrix

Using Floyd-Warshall algorithm, we compute the minimum cost to transform from any character (row) to any other character (column).

💡 How to Read This Matrix
• Row = Source character
• Column = Target character
• Cell value = Minimum cost to transform source to target
• ∞ means no path exists
• 0 on diagonal means transforming a character to itself costs nothing
